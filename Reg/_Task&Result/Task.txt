Необходимо реализовать функционал небольшого сайта со следующими страницами.

1. Главная. 

Две формы. Одна с возможностью зарегистрировать нового пользователя. Вторая для входа на сайт.
При регистрации отправляется письмо со ссылкой. Переходишь по ссылке - подтверждаешь регистрацию.
При корректном логине пользователь переходит на свой личный кабинет.

Под формами выводится список последних зарегистрированных пользователей. При клике на пользователя - переход на его профиль.

2. Личный кабинет пользователя. 

Выводится информация о пользователе. Есть возможность поменять свои ФИО, адрес и номер телефона.
Так же выводится история входов пользователя на сайт. Формат логов: дата, IP, браузер пользователя.

3. Профиль пользователя.

Выводится информация о пользователе. Ниже - история его логирования на сайт.

Реализация описанного выше функционала должна быть сделана на PHP + MySQL + JS.
Из фреймворков (при необходимости) можно использовать только jQuery. 

Результат работы надо предоставить в виде скриптов и дампа БД в архиве с минимальной документацией о функционале проекта.
